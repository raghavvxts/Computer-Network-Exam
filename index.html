Detailed Topic Breakdown
01
Introduction
Network, node, communication channel definitions
Criteria: Performance, Reliability, Scalability, Security
Data comm: Delivery, Timeliness, Accuracy
Point-to-point vs Multipoint connections
Transmission modes: Simplex / Half-duplex / Full-duplex
Guided media: UTP (Cat-5/5e/6, RJ45), STP, Coaxial, Fiber
Unguided media: Radio, microwave, infrared
02
Devices & Topologies
NIC (MAC address), Repeater (L1), Hub (L1 broadcast)
Bridge (L2, 2-port, MAC filter), Switch (L2, multiport)
Router (L3/L4), Gateway (up to L5), Firewall
Network types: PAN → LAN → MAN → WAN
Bus, Ring (token), Star, Mesh, Tree, Hybrid topologies
Bus: cables = N+1; Star: central device SPOF
03
OSI & TCP/IP Models
Why layered: divide-and-conquer, modularity, testability
OSI 7 layers: Physical→Data Link→Network→Transport→Session→Presentation→Application
Transport layer = Heart of OSI · PDU = Segment · TCP/UDP
Network layer: IP, routing, logical addressing
Data Link: MAC, frames; Physical: bits, signals
TCP/IP 4 layers: Application / Transport / Internet / Network Access
04
Physical Layer
Line coding: digital data → digital signals, formula S = cN/r
Unipolar: NRZ (1=+V, 0=0V), RZ (returns to 0 mid-bit)
Polar: NRZ-L (level), NRZ-I (inversion on 1), RZ (3 levels)
Manchester (H→L=1) used in Ethernet; always r=1/2 so baud=2×bitrate
Bipolar: AMI (0=0V, 1s alternate +/-V), Pseudoternary (reverse)
Switching: Circuit / Packet / Message; Ingress / Egress
05
Data Link Layer
Framing: Char count, char stuffing, bit stuffing (add 0 after five 1s), flags
Physical Addressing: MAC = 6 bytes / 48-bit / 12 hex / burned in NIC
Flow Control: Rate coordination sender↔receiver
Error Control: ACK, NACK, timers, sequence numbers
Access Control: Prevent/handle collisions on shared medium
Switch: CAM table (~300s), Learning, Unicast/Flooding, STP for loop avoidance
06
Error Detection & CRC
Single-bit error (rare) vs Burst error (common)
Detection (ARQ) vs Correction (FEC)
Block coding: n = k + r; 2^k valid, 2^n total codewords
Hamming distance: XOR → count 1s; d_min ≥ s+1 detect; d_min ≥ 2t+1 correct
CRC: Append r zeros → Modulo-2 divide by P → remainder = CRC
Checksum: 1's complement sum; receiver adds all → 0 = no error
07
Flow Control & ARQ
Delays: Tt=L/B, Tp=D/V, Queuing, Processing; Total = Tt+Tp+TQ+Tpro
Stop-and-Wait: Send 1 wait for ACK; η=1/(1+2a); min seq=2
Go-Back-N: Sender=N, Receiver=1; on error retransmit window; seq=N+1
Selective Repeat: Both windows=N; only retransmit lost; seq=2N; NACK
GBN & SR efficiency: η = N/(1+2a); throughput = η × B
SR window = 2^(n-1); GBN window = 2^n − 1
08
Ethernet & Bluetooth
Created 1973, Xerox PARC, Robert Metcalfe
Fast (100 Mbps), Gigabit (1 Gbps), 10G (CAT6a/fiber)
Frame: Preamble→SFD→Dest MAC→Src MAC→Length→Data→CRC
Bluetooth: 2.4 GHz ISM, 1600 hops/sec, 10m range, 1 Mb/s
Piconet: 1 master + 7 slaves; Scatternet: piconets linked
Founded 1998: Ericsson, Intel, IBM, Nokia, Toshiba
09
Wi-Fi / WLAN
802.11b: 11 Mbps / 2.4 GHz; 802.11a: 54 Mbps / 5 GHz; 802.11g: 15-20 Mbps
Ad hoc (IBSS): direct device-to-device, no AP, temporary
Infrastructure (BSS/ESS): Access Point controls traffic, supports roaming
Components: STA, BSS, AP, Portal, Distribution System, ESS
Security: WEP (broken) → WPA (TKIP) → WPA2 (AES) strongest
WEP broken: 40-bit key, static, weak IV — crackable in <10 min
10
IPv4 Addressing
32-bit, dotted-decimal, 2³² ≈ 4.3B addresses; managed by IANA/ARIN
Class A (0, /8), B (10, /16), C (110, /24), D (multicast), E (reserved)
CIDR /n: First = set host bits 0; Last = set host bits 1
Total addresses = 2^(32-n); Usable = 2^(32-n) − 2
Subnet mask shortcuts: /25→128, /26→192, /27→224, /28→240
Subnetting: bits borrowed = new-old prefix; subnets = 2^borrowed
11
IPv4 Datagram & Fragmentation
Header min 20 bytes (HLEN=5), max 60 bytes (HLEN=15)
Header size = HLEN×4; Options = (HLEN×4)−20; Data = TotalLen−Header
TTL: decremented per hop, discard at 0; Protocol: 1=ICMP, 6=TCP, 17=UDP
DF=1: don't fragment (drop + ICMP error); MF=1: more fragments follow
Fragment offset in 8-byte units; First byte = offset × 8
Reassembly ONLY at destination; fragments share same Identification
12
Network Protocols
ICMP errors: Dest unreachable, Source quench, Time exceeded, Redirect, Parameter problem
ICMP query: Echo req/reply (ping), Timestamp req/reply
ICMP DDoS: ICMP Flood, Smurf attack, Ping of Death
ARP: IP→MAC; broadcast req, unicast reply, stored in ARP cache
Proxy ARP (router replies for other subnet), Gratuitous ARP (boot broadcast, IP conflict detection)
RARP (MAC→IP, legacy), Inverse ARP (virtual circuit→IP, Frame Relay/ATM)
DHCP: DORA — Discover → Offer → Request → Acknowledge
13
Routing Fundamentals
Forwarding (per-packet, local) vs Routing (table-building, global)
Routing methods: next-hop, network-specific, default route
LPM: longest subnet mask wins when multiple routes match
AS (Autonomous System): single admin domain; Intra-AS: RIP/OSPF; Inter-AS: BGP
DVR (RIP): Bellman-Ford; share table with neighbors; count-to-infinity problem
Link State (OSPF): flood LSAs to all; Dijkstra SPF; faster convergence; no loops
14
★ Numerical Problem Types
CRC: Append r zeros → XOR long divide by P → remainder = CRC → verify
Hamming: XOR → count 1s → d_min → detection/correction capability
Stop-and-Wait: Tt=L/B, Tp=RTT/2, a=Tp/Tt, η=1/(1+2a)
GBN/SR: η=N/(1+2a), Throughput=η×B, seq numbers, window sizes
IPv4 Header: HLEN×4=header size, options=header-20, parse hex bytes
Fragmentation: Frags=⌈data/(MTU-20)⌉, offset=byte/8, MF/DF flags
Subnetting: /n → mask → first/last address → usable hosts
Baud/Nyquist/Shannon: S=cN/r, Nyquist=2B·log₂L, Shannon=B·log₂(1+SNR)
